This is just simple test for Git and GitHub.

In this project we learn about local use of git and push the local Git project to GitHub. We learn commands like -

  • git init
  • git branch -m main
  • git add .
  • git log
  • git commit -m "description message"
  • git status
  • git remote add origin [url]
  • git push --set-upstream origin main
  • git push

There are lots of commands for Git and GitHub. It's not always possible to memorize them. There are lots of information on Git commands. Best is the git's own manual. I am sharing git documentation link and a cheat sheet for git commands.

  • Git documentation
  • Git Cheat sheet

Git is mainly command base, but there are other ways to use Git like Git Desktop and Git extension/plugin on various code editors. We also learn how to make a repository in GitHub, how to make repository public or private and delete branch or repository. How to connect to Netlify and publish the project there.
